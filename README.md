# Estudiar las Rutas en ExpressJS
Este proyecto es una práctica de la asignatura _Desarrollo de Sistemas Informáticos_ del grado de Ingeniería Informática de la Universidad de La Laguna, en la que crearemos un libro en Github (utilizando gh-pages) y Gitbook donde describiremos el uso de las **rutas en ExpressJS**.
